Setting up AliEAS

AliEAS is an emergency alert system relay bot for your Discord server. You set it up using the zip code of your local area, and it broadcasts emergency alerts, warnings, and other critical information directly to your server's voice channel.

This tutorial covers setting up AliEAS on your own on-premises instance (e.g., a dedicated server or a standard computer). To set it up as a Discord administrator, join a voice channel and use the command fco!setup <zipcode>. Type fco!help for more information on the Discord side.

What You'll Need

  • Microsoft Windows
  • Python (Latest version)
  • Git (for cloning the project)
  • Credentials from the Discord Developer Portal
  • Balabolka and balcon.exe for SAPI5 voices
  • An active internet connection

Get Started

  1. Create a directory for AliEAS: 
    mkdir alieas & cd alieas
  2. Clone the project: 
    git clone https://github.com/averlice/alieas .
  3. Install dependencies: 
    pip install -r requirements.txt
  4. Configure your .env file:

    Open notepad .env and fill in your credentials:

    # Replace this with your bot token from the Discord Developer Portal
DISCORD_TOKEN=token

# Put the ID of the Voice Channel you want the bot to auto-join (Optional)
VC_CHANNEL_ID=1234567890987654

# Discord OAuth2 Credentials
DISCORD_CLIENT_ID=12345678987654321
DISCORD_CLIENT_SECRET=ThisIsASecret
REDIRECT_URI=uri
  5. Run the bot: 
    python bot.py

Congratulations! You now have an emergency alert Discord bot running on your server. Note: The default prefix is fco!.
