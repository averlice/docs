Setting up Galacticord on Windows

This tutorial will apply to running Galacticord in your local and/or cloud Windows environment. While the setup process is largely the same as Galacticord for Linux, there are differences.

What You'll Need

  • Node.js and npm installed.
  • The credentials found in the main Galacticord setup (Discord Token, Owner IDs, etc.).

Very Important: Administrator Required

Regardless of which setup script you decide to run (setup.bat or setup.ps1), you must have Administrator permissions on your Windows environment.

Other Important Notes

While largely the same as the Linux version, the main difference is that Piper will not be available for the Windows build. This is due to distortion problems when streaming Piper into the voice channel on Windows.

Setting Up

  1. Open Terminal as Administrator:

    Press Windows + R, type cmd.exe (or powershell.exe), and run it as Administrator.

  2. Run Setup Commands:

    The following command will create a folder, clone the Windows branch, and run the script all at once:

    mkdir galacticord-win64 & cd galacticord-win64 & git clone https://github.com/g4p-studios/galacticord . -b windows & .\setup.bat
  3. Wait for Installation:

    Give it 5–10 minutes to install. If prompted for Administrator permissions during the script, select Yes; it will not work otherwise.

  4. Install:

    Select option 1 (Install). This should be quick since Piper is not available.

  5. Credentials:

    Follow the prompts for Discord API keys and other keys. Press Enter to skip any you don't have; you can edit the .env file later.

Running the Bot

You have two options for running the bot, similar to the Linux script.

Option 1: Running in the Terminal

In the setup script, press 2 to run the bot and see logs in real-time.

Option 2: Running in the Background

Stop any running terminal process first.

  1. Run the setup command: 
    .\setup.bat
  2. Grant Administrator permissions when prompted.
  3. Select option 3 to create a Windows service.
  4. Give it a name (e.g., gal) and press Enter.
  5. Verify: Press Start, type Services, and look for your service name.

    If it's there, congratulations! You have a bot running in the background. Go brag about it to your parents—I'm sure they'll be impressed!
